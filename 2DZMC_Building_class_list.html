On this page you can find all the building class names..
These are found on the bottom line of the description just under class name.
You will want to add these names into the json files to have the buildings spawn im.
But you can also have these building spawn in as random events around the map if you wish.
For (E.g) If you want the building fixed into your server you would call it in via Object spawner using Json files and codes to do so.
If you want to have these spawning in as events you would use XML files (cfgeventpawns.xml)-(events.xml) to call these in...
